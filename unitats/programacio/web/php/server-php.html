PHP Server Create LAMP A LAMP server is one we use to web developing, and each letter belongs to an application we need to install:
  • *Linux*: Operating system
  • *Apache2*: Web server
  • *MySQL*: Database manager
  • *PHP*: Programming language
    •  For that we must create a Virtual Machine with Linux Mint operating system with Mate edition. You can download from [here](https://linuxmint.com/download.php). Once we have installed the server we must install with (apt install) apache2, php, mysql-server, mysql-client and openssh-server also.     Set private and public key (in your computer) We need to access to server through ssh and with our public and private key. First of all we must create the keys: ssh-keygen -t rsa There ask us for name file (we can type our name) and a passphrase that is a password. After that we have two files, one private key with name file we typed and another one with the same name but with pub extensions. Now we must copy public key to the server: ssh-copy-id -i filename.pub serveruser@serverip Visual Studio Code configuration Tasks we need to do are the following:
  • Install Remote SSH Extension
  • Set file configuration:
  • Press F1
  • Type: Remote SSH: Open SSH Configuration File
  • Set the file: Host server HostName your_server_ip User server_user IdentityFile where_is_your_private_key
  • Connection to server:
  • Press Open a Remote Window Button (at the bottom left)
  • Select Connect to host
  • Select server
  • Type password
    •      Set the server folder to developing We must to create a folder where we're going to develop:
  • Go to /var/www/html
  • Create a folder with name php
  • Create a file index.php inside php folder
  • Set owner's folder sudo chown -R serveruser:serveruser /var/www/html/php
    • Access to web folder In VSC open the folder /var/www/html/php and edit index.php to create a html structure with a text in body that say Hello! After saving the file, you can go to a web browser and find server_ip/php and you'll look Hello message.
