  • Create a calendar.php file
  • First of all we're going to create a h1 with title Calendar of 2023
  • We must create a div per each month and inside an h2 and a div per each day. But first we are going to create only the h2 with the name of the month. To do that create a function called create_month with one parameter $month. Add a loop to call that function from month 1 until 12. Inside the function show the number of the month.
  • Then we need to show the name of the month. Using the following code we can get the name. //Save in a new variable $complete_month the month with two digits //So if month is less than 10, add '0' before the month in $complete_month //else $complete_month is the value of the month //Create a variable $date_string that must contain the first day of these month and the year using this format yyyy-mm-dd //Save in $month_word the month with letter $month_word = date('F', strtotime($date_string)); //Show an h2 with the variable month_word
  • Now we want to show each day of each month. For that create_days with parameter $month. In create_month function call create_days fuction sending the $month parameter. Inside this new function you must show from 1 until the number days of the months.
