Marvel exercise We're going to create a website with a list of marvel's superheroes. Follow this steps to reproduce the image
  • Create a marvel.php file
  • Inside, with code php create a function create_superhero with 3 parameters: name, colors, and power
  • The function must create and show a div with class superhero. Inside this div you'll create a h1, h2 and h3 tag.
  • After that call the function 5 times with different data
  • The container of all divs must have these styles for see in mode grid: display: grid; column-gap: 20px; row-gap: 20px; grid-template-columns: auto auto auto; justify-content: space-around;
