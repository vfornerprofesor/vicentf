Responsive design El disseny responsiu és una tècnica de disseny web que permet que una pàgina web s'adapti a diferents mides de pantalla i dispositius, com ara ordinadors, tauletes i mòbils. Això es fa utilitzant CSS per ajustar el disseny i el contingut en funció de la mida de la pantalla. Ús de les media queries Les media queries són una part fonamental del disseny responsiu. Permeten aplicar diferents estils CSS en funció de les característiques del dispositiu, com la mida de la pantalla, la resolució, l'orientació, etc. Exemple de disseny responsiu amb media queries A continuació es mostra un exemple de com utilitzar les media queries per crear un disseny responsiu que s'adapti a diferents mides de pantalla: El codi base seria el següent que mostraria 3 caixes en vertical i volem mostrar en pantalla d'ordinador les caixes en 3 columnes, en tauleta 2 columnes i en mòbil 1 columna. <div id="contenedor"> <div id="uno">1</div> <div id="dos">2</div> <div id="tres">3</div> </div> #contenedor { margin: 10px; display: grid; border: 1px solid grey; padding: 10px; } #uno, #dos, #tres { width: 300px; height: 300px; padding: 5px; font-size: 50px; line-height: 300px; text-align: center; margin-inline: auto; } #uno { background-color: coral; } #dos { background-color: cadetblue; } #tres { background-color: lightgreen; } Ara, per aplicar el que volem fer amb les columnes hem d'indicar els estils en els diferents models: /* Computer styles */ @media (min-width: 1024px) { #contenedor { grid-template-columns: 1fr 1fr 1fr; } } /* Tablet styles */ @media (min-width: 768px) and (max-width: 1023px) { #contenedor { grid-template-columns: 1fr 1fr; } } /* Phone styles */ @media (max-width: 767px) { #contenedor { grid-template-columns: 1fr; } } En aquest exemple, hem canviat els estils del contenedor que té les 3 caixes per tal que tinga uns valors per a la propiedad grid-template-columns. Amb aquesta configuració, tal i com s'obriga el web en diferents dispositius es veurà les diferents configuracions. Exercici 1 Crea una pàgina web que tinga una imatge (gran) d'una empresa i baix el títol amb el nom en versió pantalla d'ordinador; en tauleta i mòbil estaran la imatge i el títol un al costat de l'altre. Has de tindre en compte el tamany de la imatge en versió tauleta i en versió mòbil. Exercici 2 Agafa l'exercici que has fet abans dels països i fes que es mostren 3 columnes en mode pantalla d'ordinador; 2 en mode tauleta; i 1 en mode mòbil. Exercici 3 Crea un formulari senzill d'inici de sessió amb el nom de la teua empresa, un camp per a l'usuari o el correu i un altre camp per a la contrasenya. En mode pantalla d'ordinador ha d'estar l'etiqueta del camp al costat del input; en mode tauleta el que tu consideres; en mode mòbil haurà d'estar l'etiqueta dalt i el camp baix.
