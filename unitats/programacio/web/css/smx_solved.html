Sistemes Microinformàtics i Xarxes

Vicent Forner

2n curs - 24/25

Seguretat informàtica

La seguretat informàtica és essencial per protegir els sistemes i les dades de les amenaces externes. En aquest mòdul aprendràs a identificar i mitigar els riscos per a la seguretat dels sistemes informàtics. Estudiaràs els diferents tipus d'atacs informàtics, com ara els virus, els troians i els atacs de denegació de servei, i aprendràs a implementar mesures de seguretat per prevenir-los. A més a més, exploraràs conceptes com ara els firewalls, els sistemes de detecció d'intrusos i la criptografia.

Sistemes Operatius en Xarxa

Un sistema operatiu en xarxa és un programari que gestiona els recursos d'un ordinador i permet la comunicació amb altres ordinadors connectats a la xarxa. Aquest mòdul t'ensenyarà a configurar i gestionar sistemes operatius en xarxa, com ara Windows Server i Linux. Aprendràs a crear i gestionar comptes d'usuari, a compartir recursos entre diferents ordinadors i a configurar els serveis de xarxa. A més a més, exploraràs conceptes com ara els dominis, els grups de treball i els protocols de xarxa.

Aplicacions web

En aquest mòdul aprendràs a desenvolupar i gestionar aplicacions web. Les aplicacions web són programes que s'executen en un servidor i es visualitzen a través d'un navegador web. Aquestes aplicacions són la base de molts serveis en línia que utilitzem diàriament, com ara les xarxes socials, els comerços electrònics i els correus electrònics. Estudiaràs els llenguatges de programació utilitzats per crear aplicacions web, com ara HTML, CSS i JavaScript, i aprendràs a dissenyar interfícies d'usuari intuïtives i funcionals. A més a més, exploraràs conceptes com ara bases de dades, seguretat web i optimització del rendiment.

Serveis en Xarxa

Els serveis en xarxa són programes que s'executen en un servidor i proporcionen funcions específiques a altres dispositius connectats a la xarxa. Aquest mòdul t'ensenyarà a configurar i gestionar diversos serveis en xarxa, com ara servidors de correu electrònic, servidors web, servidors FTP i servidors de bases de dades. Aprendràs a configurar els protocols de comunicació necessaris per a cada servei i a assegurar la seva disponibilitat i seguretat. A més a més, exploraràs conceptes com ara la direcció IP, el DNS i els ports de comunicació.
