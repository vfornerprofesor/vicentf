Exercicis CSS - Grup de música Has de reproduir la següent imatge: *Pistes per a l'estructura*
  • Caixa principal: amb imatge i títol 1
  • Caixa menú: dos enllaços
  • Imatge principal
  • Títol 2 per a galeria d'imatges i caixa galeria amb 7 imatges
  • Títol 2 per a millors cançons i dos caixes dins cadascuna amb títol 3, video de Youtube i etiqueta pre
  • Per obtindre el vídeo de Youtube has d'anar al video de Youtube, donar-li a compartir i copiar el codi que apareix en l'apartat *insertar*.
    •      *Pistes per als estils*
  • Etiqueta html font-family: sans-serif; scroll-behavior: smooth;
  • Caixa menú: mirar en teoria css com posar-los en flex i al centre.
  • Caixa galeria: mirar en teoria css com posar-los en flex wrap.
  • Caixa millors cançons: mirar en teoria css com posar-los en grid i dos columnes.
  • Enllaços menú: al passar el cursor per damunt que s'intercanvien el color de fons i el color de la lletra.
    • *Pista perquè els enllaços vajen a una secció de la pàgina web i no a un enllaç extern*
  • Afegeix un identificador al títol de galeria d'imatges i al títol de millors cançons (sinó el tenen ja)
  • Als enllaços, en l'atribut href has de posar un # seguit del identificador dels títols
