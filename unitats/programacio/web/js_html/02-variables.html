Variables en JS Què són les variables? Les variables són un concepte bàsic de tots els llenguatges de programació. En JavaScript, les variables s'utilitzen per a emmagatzemar dades. Podem utilitzar variables per a emmagatzemar qualsevol tipus de dades, com ara números, cadenes de text, o objectes. Declarar variables Per a declarar una variable en JavaScript, utilitzem la paraula clau let o const.
  • La paraula clau *let* declara una variable amb abast de bloqueig. Això significa que la variable només és visible dins del bloc en què s'ha declarat i *es podrà modificar*
  • La paraula clau *const* declara una variable amb abast de bloc i valor constant. Això significa que la variable només és visible dins del bloc en què s'ha declarat i que *el seu valor no es pot canviar*.
    • let nom = "Joan"; const EDAT = 30; En aquest exemple, hem declarat dues variables:
  • La variable *nom* és una variable amb abast de bloqueig que emmagatzema una cadena de text.
  • La variable *EDAT* és una variable amb abast de bloqueig i valor CONSTANT que emmagatzema un número.
    • Assignació de valors i accés a les variables Una vegada que hem declarat una variable, podem assignar-li un valor.
