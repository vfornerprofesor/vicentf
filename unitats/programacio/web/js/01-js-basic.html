Conceptes bàsics de JS Què és JS? JavaScript és un llenguatge de programació basat en objectes (també conegut com a script) que s'utilitza per a crear aplicacions web interactives. És un llenguatge essencial per a qualsevol desenvolupador web, ja que permet afegir funcionalitats a les pàgines web que no es poden crear només amb HTML i CSS. JavaScript s'executa al costat del navegador web, cosa que significa que no cal instal·lar cap programari addicional per a utilitzar-lo. Això el converteix en una opció molt accessible per a desenvolupar aplicacions web. Amb JavaScript pots fer coses com:
  • Afegir interacció a les pàgines web, com ara fer que els elements s'animen, reaccionen a les accions de l'usuari o mostren informació dinàmica.
  • Validar les dades introduïdes per l'usuari.
  • Crear efectes visuals.
  • Comunicar-te amb servidors web.
    •  JavaScript és un llenguatge molt potent i flexible, i les seves possibilitats són gairebé il·limitades.     Afegint codi JS a una pàgina web El codi JS el podem posar tant dins del fitxer HTML com en un fitxer extern. Dins de l'HTML L'etiqueta utilitzada és script i dins d'ella s'escriu tot el codi. Fíxa't en els dos codis d'exemple per entendre què és el que s'executa abans <body> <p>Hola HTML</p> <script> document.write('Hola JS'); </script> </body> Hola HTML Hola JS <body> <script> document.write('Hola JS'); </script> <p>Hola HTML</p> </body> Hola JS Hola HTML En un fitxer a banda L'etiqueta utilitzada és script però en aquest cas s'indica el nom del fitxer JS. Fíxa't en els dos codis d'exemple per entendre què és el que s'executa abans <body> <p>Hola HTML</p> <script src="script.js"></script> </body> //script.js document.write('Hola JS'); Hola HTML Hola JS <body> <script src="script.js"></script> <p>Hola HTML</p> </body> //script.js document.write('Hola JS'); Hola JS Hola HTML Mostrant missatges en pantalla i en consola Com has vist en el codi anterior amb document.write podem escriure el que vulga'm en pantalla. Anem a veure diferents exemples: document.write("Hola JS només amb text"); document.write("<h1>Hola JS dins d'un títol h1</h1>"); document.write("<p>Hola JS dins d'un paràgraf</p>"); També podem escriure missatges per a nosaltres, els programadors web, per tal d'anar comprovant coses i que no es vegen directament en pantalla. Per això utilitzem el *console.log()* i es veu amb la ferramenta web que s'obri al pressionar F12 al navegador. console.log('Hola'); console.log('Adeu'); Comentant codi JS Per comentar codi i fer anotacions sense que es mostren per pantalla tenim dos opcions: //Aquest és un comentari d'una linia //Aquest és un comentari també d'una linia //document.write('hola'); //La linia anterior no s'executa /* Aquest és un comentari de varies línies fins que no pose els mateixos símbols al final seguirà sent un comentari i per tant, aquesta linia: document.write('hola'); no s'executaria */
