Comences pel primer, el segon... Si trobes el llibre en la posició 10, té sentit continuar mirant els 90 llibres restants? No. En eixe moment, deixes de buscar. En programació, el break és eixa "eixida d'emergència" que trenca el bucle immediatament i continua amb la resta del programa. Anem a vore-ho amb codi senzill, ja que estem començant i encara no hem vist estructures complexes de dades. 1. Exemple amb un bucle for Imagina que volem trobar el número 7 entre els números del 0 al 9. console.log("Comencem a buscar el número 7..."); for (let i = 0; i < 10; i++) { console.log("Mirem el número: " + i); if (i === 7) { console.log("Ja l'hem trobat! Eixim del bucle."); break; // Aci trenquem el bucle. Els números 8 i 9 mai es mostraran. } } console.log("Continuem amb la resta del programa."); Què passa ací? Sense el break, el programa imprimiria fins al 9. Amb el break, quan i val 7, el bucle es deté i salta fora. Això forma part de les "estructures de salt" que tenim al temari. 2.
