Objectes (o diccionaris) Què són els objectes i per a què serveixen? Les objectes són una estructura de dades que permeten emmagatzemar múltiples valors en una sola variable. Són com una cistella on puc guardar diversos objectes. Emmagatzematge Múltiple Permeten guardar diversos valors en una sola variable però amb diferents noms. Accés Ràpid Permeten accedir a qualsevol element de la llista utilitzant un nom (com un identificador). Flexibilitat Són flexibles i es poden utilitzar per diversos propòsits, com emmagatzemar dades, realitzar càlculs, etc. Com definir i utilitzar un objecte Per crear un objecte en JavaScript, podem utilitzar el constructor Object o la notació literal. let objecte = { nom: "pera", color: "verd", preu: 1.5 }; Per accedir a un element de l'objecte, podem utilitzar el seu nom. let objecte = { nom: "pera", color: "verd", preu: 1.5 }; let nom = objecte.nom; //pera let color = objecte.color; //verd let total = objecte.preu; //1.5 Operacions amb objectes Afegir/Canviar elements Per afegir un element a l'objecte o canviar el valor, podem utilitzar el seu nom. let fruita = { nom: "pera", color: "verd", preu: 1.5 }; fruita.nom = "platan"; //Canvia el nom de la pera per platan fruita.color = "groc"; //Canvia el color de la pera per groc fruita.preu = 2.5; //Canvia el preu del platan per 2.5 fruita.procedencia = "Comunitat Valenciana"; //Afegeix la procedència del platan Eliminar elements Per eliminar un element de l'objecte, podem utilitzar el mètode delete. let fruita = { nom: "pera", color: "verd", preu: 1.5 }; delete fruita.nom; //Elimina el nom de la pera Longitud de l'objecte Per obtenir la longitud de l'objecte, podem utilitzar la propietat length. let fruita = { nom: "pera", color: "verd", preu: 1.5 }; fruita.length; //3 Accés a tots els elements Per accedir a tots els elements de l'objecte, podem utilitzar un bucle for. let fruita = { nom: "pera", color: "verd", preu: 1.5 }; for (let element in fruita) { console.log(element + ": " + fruita[element]); } Quines propietats té? Per obtenir les propietats d'un objecte, podem utilitzar el mètode Object.keys(). let fruita = { nom: "pera", color: "verd", preu: 1.5 }; let propietats = Object.keys(fruita); console.log(propietats); // ["nom", "color", "preu"] Què valen? Per obtenir els valors d'un objecte, podem utilitzar el mètode Object.values(). let fruita = { nom: "pera", color: "verd", preu: 1.5 }; let valors = Object.values(fruita); console.log(valors); // ["pera", "verd", 1.5] Mostrar tot l'objecte amb stringify Per mostrar tot l'objecte, podem utilitzar el mètode JSON.stringify(). let fruita = { nom: "pera", color: "verd", preu: 1.5 }; let json = JSON.stringify(fruita); console.log(json); // {"nom":"pera","color":"verd","preu":1.5} Exercicis Exercici 1 Crea un objecte JavaScript anomenat llibre que emmagatzeme el títol, l'autor i l'any de publicació. Crea una funció que permeta a l'usuari afegir una nova propietat (la ISBN) a l'objecte i mostrar el contingut actualitzat. Aquesta operació demostrarà com afegir una nova propietat a un diccionari Exercici 2 Crea un objecte JavaScript anomenat compte que guarde un nom d'usuari i el seu saldo (saldo). Crea una funció que simule un ingrés (suma) i una altra que simule una retirada (resta). Ambdues funcions han de modificar el valor de la propietat saldo de l'objecte Exercici 3 Crea un objecte JavaScript anomenat empleat amb les propietats: nom, departament, i telefon. Aquest objecte es crea amb la informació completa. Crea una funció que, en ser executada, esborre la propietat telefon de l'objecte, simulant que l'empleat passa a teletreballar i el telèfon de l'oficina ja no és rellevant. Mostra l'estat de l'objecte abans i després de l'esborrament. Exercici 4 Amplia l'exercici del llibre. Crea un objecte llibre. En lloc d'afegir una propietat fixa (ISBN), permet a l'usuari introduir el nom de la nova propietat (clau) i el seu valor. La funció ha d'afegir aquesta parella clau-valor a l'objecte i després redibuixar tot l'objecte a la pantalla, utilitzant funcions del DOM per crear un nou div amb paràgrafs per a cada propietat Exercici 5 Crea una llista d'objectes (un diccionari on les claus són els noms d'usuari) que emmagatzeme diversos comptes amb els seus respectius saldos. A la interfície, l'usuari ha de poder seleccionar un dels comptes mitjançant un desplegable. En introduir una quantitat i fer clic a "Ingressar", el saldo només es suma al compte seleccionat. La visualització s'ha de fer creant un div nou per a cada usuari, mostrant el seu nom i saldo actualitzat. Exercici 6 Basant-se en l'Exercici 5, ara cada objecte d'usuari ha de guardar, a més del saldo actual, una llista (operacions) amb els ingressos realitzats. En fer un ingrés, s'ha de sumar el saldo i afegir la quantitat a aquesta llista. La visualització ha de complir:
  • 1. Mostrar l'estat actual de cada compte (Nom i Saldo).
  • 2. Per a cada compte, mostrar la llista de totes les operacions d'ingrés realitzades.
  • 3. Calcular i mostrar un Gran Total de tots els saldos combinats.
