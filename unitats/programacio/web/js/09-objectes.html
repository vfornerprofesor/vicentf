Objectes (o diccionaris) Què són els objectes i per a què serveixen? Les objectes són una estructura de dades que permeten emmagatzemar múltiples valors en una sola variable. Són com una cistella on puc guardar diversos objectes. Emmagatzematge Múltiple Permeten guardar diversos valors en una sola variable però amb diferents noms. Accés Ràpid Permeten accedir a qualsevol element de la llista utilitzant un nom (com un identificador). Flexibilitat Són flexibles i es poden utilitzar per diversos propòsits, com emmagatzemar dades, realitzar càlculs, etc. Com definir i utilitzar un objecte Per crear un objecte en JavaScript, podem utilitzar el constructor Object o la notació literal. let objecte = { nom: "pera", color: "verd", preu: 1.5 }; Per accedir a un element de l'objecte, podem utilitzar el seu nom. let objecte = { nom: "pera", color: "verd", preu: 1.5 }; let nom = objecte.nom; //pera let color = objecte.color; //verd let total = objecte.preu; //1.5 Operacions amb objectes Afegir/Canviar elements Per afegir un element a l'objecte o canviar el valor, podem utilitzar el seu nom. let fruita = { nom: "pera", color: "verd", preu: 1.5 }; fruita.nom = "platan"; //Canvia el nom de la pera per platan fruita.color = "groc"; //Canvia el color de la pera per groc fruita.preu = 2.5; //Canvia el preu del platan per 2.5 fruita.procedencia = "Comunitat Valenciana"; //Afegeix la procedència del platan Eliminar elements Per eliminar un element de l'objecte, podem utilitzar el mètode delete. let fruita = { nom: "pera", color: "verd", preu: 1.5 }; delete fruita.nom; //Elimina el nom de la pera Longitud de l'objecte Per obtenir la longitud de l'objecte, podem utilitzar la propietat length. let fruita = { nom: "pera", color: "verd", preu: 1.5 }; fruita.length; //3 Accés a tots els elements Per accedir a tots els elements de l'objecte, podem utilitzar un bucle for. let fruita = { nom: "pera", color: "verd", preu: 1.5 }; for (let element in fruita) { console.log(element + ": " + fruita[element]); } Quines propietats té? Per obtenir les propietats d'un objecte, podem utilitzar el mètode Object.keys(). let fruita = { nom: "pera", color: "verd", preu: 1.5 }; let propietats = Object.keys(fruita); console.log(propietats); // ["nom", "color", "preu"]
