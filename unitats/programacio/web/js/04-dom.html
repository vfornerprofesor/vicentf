DOM Què és el DOM i què podem fer? El DOM és el Document Object Model que ens permet agafar tots els elements d'un web i canviar el contingut, la estructura i l'estil. És a dir ens permet modificar elements com el text d'un paràgraf, afegir elements HTML dins d'altres o també canviar el color d'alguns. Agafar elements <body> <h1 id="titol_hola">Hola!</h1> <p id="p_comestas">Com estàs?</p> </body> //Agafar el títol i guardar-lo en una variable //S'agafa utilitzant document.getElementById //i enviant-li el identificador del element que volem agafar let titol = document.getElementById('titol_hola'); //Agafar el paràgraf i guardar-lo en una variable let paragraf = document.getElementById('p_comestas'); Agafar valors dels inputs Per recollir informació que posa l'usuari que interactua amb el web utilitzem els inputs, per tant, dels elements HTML hem d'obtindre el valor que han escrit de la següent manera: <input id="nom" type="text"> let inputNom = document.getElementById('nom'); let nom = inputNom.value; Canviar text <body> <h1 id="titol1" onclick="canviarTitol1()">Titol 1</h1> <h2 id="titol2" onclick="canviarTitol2()">Titol 2</h2> <p id="paragraf" onclick="canviarParagraf()">Paragraf</p> </body> //Agafe el títol 1 i li canvie el text a 'HOLA!' function canviarTitol1() { let titol1 = document.getElementById('titol1'); titol1.textContent = 'HOLA!'; } //Agafe el títol 2 i li canvie el text a 'Com estàs?' function canviarTitol2() { let titol2 = document.getElementById('titol2'); titol2.textContent = 'Com estàs?'; } //Agafe el paràgraf i li canvie el text a 'Estic aprenent JS' function canviarParagraf() { let paragraf = document.getElementById('paragraf'); paragraf.textContent = 'Estic aprenent JS'; } Fixa't que al fer clic sobre cada element s'executa la funció pertinent. Canviar estils <body> <p id="roig" onclick="canviarParagrafRoig()">Roig</p> <p id="verd" onclick="canviarParagrafVerd()">Verd</p> <p id="blau" onclick="canviarParagrafBlau()">Blau</p> </body> //Amb la propietat style podem canviar tots els estils d'un element HTML function canviarParagrafRoig() { var p = document.getElementById('roig'); p.style.color = 'red'; } Recorda que estils hi ha molts: width, height, color, backgroundColor, fontSize, border, etc. Exercicis Exercici 1 Reprodueix el següent web amb el nom de 4 companys de classe i fes que al fer clic a cadascú, es canvie el tamany i color de la lletra del seleccionat i es torne al tamany i color original a la resta.
