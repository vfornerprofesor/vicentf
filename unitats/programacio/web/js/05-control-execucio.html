Control de l'execució En els programes que escrivim a vegades hem de fer unes accions o unes altres depenent de condicions que puguen haver o també hem de repetir certes accions en tot el programa. És per això que en programació existeixen instruccions de control de l'execució que ens permeten realitzar aquestes accions. SI / SINO Quan volem executar unes accions o unes altres utilitzem el condicional if acompanyat a vegades del if/else. Anem a crear un programa per validar la nota que introduïrem en un input de tipus número: <h1>Revisió de la nota</h1> <p>Nota:</p> <input type="number" name="nota" id="nota"> <button onclick="revisarNota()">Revisar</button> <p id="resultat"></p> <script> function revisarNota() { let inputNota = document.getElementById('nota'); let nota = parseInt(inputNota.value); let resultat = document.getElementById('resultat'); //Revisar nota } </script> En el comentari Revisar nota, hem de fer que SI la nota és major o igual que 5 en resultat pose 'Has aprovat!'. Per fer-ho anem a utilitzar el condicional if: if(condició) { instruccions a executar } if(nota >= 5) { resultat.textContent = 'Has aprovat!'; } D'aquesta manera, en escriure una nota major o igual que 5 es canviarà el text de després del botó a 'Has aprovat!'. Però, i si es posa una nota inferior a 5? Hem de revisar que No es complisca la primera condició. Per això s'utilitza el 'else' que sempre ha d'anar després d'un if. if(nota >= 5) { resultat.textContent = 'Has aprovat!'; } else { //canviar el text de resultat } A més, també es poden posar ifs i elses dins d'altres if o else. Anem a controlar que la nota siga inferior a 0 o major de 10. if(nota >= 5) { resultat.textContent = 'Has aprovat!'; } else { if(nota < 0 || nota > 10) { resultat.textContent = 'La nota no és vàlida!'; } else { resultat.textContent = 'Has suspés!'; } } Acabar exercici Fes que canvie el color del text amb verd (aprovat), roig (suspés) i groc (nota invàlida). També fes que en lloc de mostrar aprovat només que mostre Suficient, Bé, Notable o Excel·lent.
