Funcions 1. Què són les Funcions i Per a Què Serveixen? Una funció és un bloc de codi reutilitzable dissenyat per realitzar una tasca concreta. Penseu-hi com una "miniprèmia" o un "robot" dins del vostre programa. Reutilització del Codi Permeten executar el mateix codi moltes vegades sense haver de reescriure'l. Si necessiteu calcular el quadrat d'un nombre deu vegades, només definiu la funció una vegada. Organització i Manteniment Fan que el codi siga més modular, net i fàcil de llegir. En lloc de tenir un codi "monolític", el dividim en tasques lògiques. Si hi ha un error en una part, sabeu exactament on buscar. Abstracció Permeten amagar la complexitat. Quan crideu a console.log(), no necessiteu saber com funciona internament, només que "escriu alguna cosa per consola". Les vostres funcions fan el mateix. 2. Com Definir i Utilitzar una Funció (Amb i Sense return) Per poder utilitzar una funció, primer l'hem de definir. A JavaScript, la sintaxi més habitual és utilitzant la paraula clau *function*. Definició Bàsica d'una Funció function nomDeLaFuncio() { // Instruccions que executa la funció console.log("Aquesta funció s'ha executat!"); } // Per utilitzar-la o "cridar-la": nomDeLaFuncio(); // S'executa el codi de dins Funcions Sense Retorn (Procediments) Una funció sense la paraula clau *return* realitza una acció (com imprimir un missatge, modificar una variable global o actualitzar l'estil d'un element), però no retorna un valor explícitament al lloc on ha estat cridada. De fet, totes les funcions a JS retornen undefined si no hi ha un return explícit. Exemple de Funció Sense return: // La funció només SALUDA (fa una acció) function saludar() { let missatge = "Hola, benvingut/da al món de les funcions!"; console.log(missatge); } // Crida: saludar(); // Només es veu el missatge per consola Funcions Amb Retorn (return) Les funcions que utilitzen *return* tenen com a objectiu calcular i retornar un valor al codi que les ha cridat. Aquest valor retornat es pot guardar en una variable o utilitzar-se directament en una expressió. *Regla Clau:* Quan s'executa la instrucció return, la funció finalitza immediatament, i el codi posterior no s'executa. Exemple de Funció Amb return: // La funció CALCULA i RETORNA un resultat function obtenirPI() { return 3.14159; // Retorna el valor de PI } // Crida i assignació del valor retornat: let elMeuPI = obtenirPI(); console.log("El valor de PI és: " + elMeuPI); // Mostra: El valor de PI és: 3.14159 // O es pot utilitzar directament: console.log(obtenirPI() * 2); // Mostra: 6.28318 (retorna el valor i es multiplica) 3. Funcions amb i sense Paràmetres Funcions Sense Paràmetres Com hem vist abans, són funcions que realitzen la seua tasca sense necessitat de cap dada externa. Exemple: function mostrarDataActual() { let avui = new Date(); // Obté la data del sistema console.log("La data és: " + avui.toDateString()); // No necessita cap dada d'entrada } mostrarDataActual(); La Importància dels Paràmetres Els paràmetres són variables que actuen com a entrades de la funció, permetent-li rebre dades de l'exterior. Són essencials per fer que la funció siga flexible i genèrica, ja que pot operar amb diferents valors cada vegada que és cridada.
  • *Definició:* Es col·loquen dins dels parèntesis de la definició de la funció.
  • *Argument:* És el valor concret que es passa a la funció quan la cridem.
    • Exemple de Funció Amb Paràmetres i return: // 'base' i 'altura' són els PARÀMETRES function calcularAreaTriangle(base, altura) { let area = (base * altura) / 2; return area; // Retorna el resultat del càlcul } // Crides amb ARGUMENTS diferents: let area1 = calcularAreaTriangle(10, 5); // Arguments: 10 i 5 let area2 = calcularAreaTriangle(20, 8); // Arguments: 20 i 8 console.log("Àrea 1: " + area1); // Mostra: 25 console.log("Àrea 2: " + area2); // Mostra: 80     Exercicis de Funcions Exercici 1: El meu perfil de presentació
  • *Enunciat:* Crea una funció que es diu *presentar* que rebi com a paràmetres el teu nom i la teva edat. Hi haurà dos inputs (nom i edat) i un botó. Al fer click al botó es recolliran els valors dels inputs i es cridarà a la funció amb els paràmetres indicats.
  • *Instruccions:* La funció ha d'afegir un títol h1 al body amb el missatge "Em dic [el teu nom] i tinc [la teua edat] anys.". Crida la funció amb les teves dades.
    • Exercici 2: Calculadora de l'IVA
  • *Enunciat:* Crea una funció que es diu *calcularPreuFinal* que rebi dos paràmetres: un preu i un iva. Hi haurà dos inputs (preu i iva) i un botó. Al fer click al botó es recolliran els valors dels inputs i es cridarà a la funció amb els paràmetres indicats.
  • *Instruccions:* La funció ha de calcular el preu final aplicant l'IVA i ha d'escriure un missatge en un paràgraf al web.
    • Exercici 3: Mostra la seqüència de números
  • *Enunciat:* Crea una funció que es diu *mostrarNumeros* que rebi com a paràmetres dos nombres i mostrarà els nombres entre el primer i el segon. Hi haurà 3 blocs amb 2 inputs i botó cadascú que executaran la mateixa funció.
  • *Instruccions:* La funció ha de mostrar un paràgraf amb els números de la seqüència.
    • Exercici 4: Mix
  • *Enunciat:* Hi haurà diferents blocs. Un amb un input per al nom i l'edat com l'exercici 1. Un amb el preu i l'iva de l'exercici 2. Un amb dos números per mostrar la seqüència com el número 3. Hauràs de fer una funció per a cada botó i a més una funció *afegeixTextAPantalla* que rebrà els paràmetres tipus i missatge. El tipus serà 'p', 'h1', 'h2' i el missatge serà el text que es mostrarà en pantalla. El primer bloc cridarà a la funció per crear un h1, el bloc dos per crear un h2 i el bloc 3 per crear un paràgraf.
  • *Instruccions:* La funció ha de rebre el tipus i el missatge. Crearà un element com el que reba segons el tipus, li canviarà el text per el missatge que reba i l'afegirà al body.
