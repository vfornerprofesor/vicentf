Llistes Què són les llistes i per a què serveixen? Les llistes són una estructura de dades que permeten emmagatzemar múltiples valors en una sola variable. Són com una cistella on puc guardar diversos objectes. Emmagatzematge Múltiple Permeten guardar diversos valors en una sola variable, en lloc d'assignar un valor per variable. Accés Ràpid Permeten accedir a qualsevol element de la llista utilitzant un índex (com un número de posició). Flexibilitat Són flexibles i es poden utilitzar per diversos propòsits, com emmagatzemar dades, realitzar càlculs, etc. Com definir i utilitzar una llista Per crear una llista en JavaScript, podem utilitzar el constructor Array o la notació literal. let llista = ["pera", "poma", "raïm"]; Per accedir a un element de la llista, podem utilitzar el seu índex. let llista = ["pera", "poma", "raïm"]; let element = llista[0]; //pera Operacions amb llistes Afegir elements Per afegir un element a la llista, podem utilitzar el mètode push. let llista = ["pera", "poma", "raïm"]; llista.push("platan"); // ["pera", "poma", "raïm", "platan"] Afegir elements al principi Per afegir un element al principi de la llista, podem utilitzar el mètode unshift. let llista = ["pera", "poma", "raïm"]; llista.unshift("platan"); // ["platan", "pera", "poma", "raïm"] Eliminar elements Per eliminar un element de la llista, podem utilitzar el mètode pop. let llista = ["pera", "poma", "raïm"]; llista.pop(); // ["pera", "poma"] Eliminar elements al principi Per eliminar un element al principi de la llista, podem utilitzar el mètode shift. let llista = ["pera", "poma", "raïm"]; llista.shift(); // ["poma", "raïm"] Longitud de la llista Per obtenir la longitud de la llista, podem utilitzar la propietat length. let llista = ["pera", "poma", "raïm"]; llista.length; // 3 Afegir elements en posició concreta Per afegir un element en una posició concreta de la llista, podem utilitzar el mètode splice. let llista = ["pera", "poma", "raïm"]; llista.splice(1, 0, "platan"); // ["pera", "platan", "poma", "raïm"] Eliminar elements en posició concreta Per eliminar un element en una posició concreta de la llista, podem utilitzar el mètode splice. let llista = ["pera", "poma", "raïm", "platan"]; llista.splice(2, 1); // ["pera", "poma", "platan"] Exercicis Exercici 1 Practica amb les operacions bàsiques de les llistes.
