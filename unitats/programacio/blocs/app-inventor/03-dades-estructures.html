Dades i estructures amb App Inventor Aquest apartat introduïrà conceptes clau per manipular dades dins les aplicacions: variables avançades, llistes i iteracions. L'objectiu és que l'alumnat pugui guardar i treballar amb col·leccions de dades, com a llistes de tasques o noms, i processe aquestes dades mitjançant recorreguts i cerques. Què són les llistes? Una llista és una col·lecció ordenada d'elements, on cada element té una posició que es denomina *índex*. A App Inventor, els índexs comencen en 1 (el primer element). Aquestes dades poden ser text, números, o fins i tot altres llistes o variables. Per exemple, si la llista és ["manzana", "plátano", "naranja"], "manzana" està a l'índex 1. Com crear i manipular llistes? 1. Crear una llista En els blocs, "make a list" permet generar una llista amb diversos elements. Es pot modificar afegint o eliminant elements dins. 2. Afegir elements Es pot usar el bloc "add items to list" per afegir nous elements a una llista existent. 3. Llegir elements Amb el bloc "select list item" es pot accedir a un element concret mitjançant l'índex. 4. Iterar llista El bloc "for each item in list" permet recórrer tots els elements d'una llista i executar accions per cada un. Variables avançades És important saber que les llistes són un tipus especial de variable que permet guardar múltiples valors relacionats. Crear variables globals o locals per emmagatzemar aquestes llistes facilitarà la gestió de dades. Exercicis pràctics Exercici 1: Llista de tasques pendents
  • Fes una app que permeti escriure tasques i afegir-les a una llista.
  • Mostra el nombre total de tasques pendents.
  • Permet eliminar tasques de la llista.
