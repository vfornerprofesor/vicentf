Models i protocols de comunicació TEORIA TCP: El protocolo de control de transmisión (TCP) es un protocolo de transporte orientado a la conexión que garantiza la entrega de los datos a su destino. IP: El protocolo de internet (IP) es un protocolo de red que asigna direcciones a los dispositivos en una red. HTTP: El protocolo de transferencia de hipertexto (HTTP) es un protocolo de aplicación que se utiliza para transferir datos a través de internet. FTP: El protocolo de transferencia de archivos (FTP) es un protocolo de aplicación que se utiliza para transferir archivos a través de internet. SMTP: El protocolo simple de transferencia de correo electrónico (SMTP) es un protocolo de aplicación que se utiliza para enviar correo electrónico. PRÀCTICA Joc de memory?
