Activitat 4 - Instal·lar un sistema operatiu Anem a instal·lar un sistema operatiu dins de l'ordinador del centre utilitzant el programa Virtual Box. Aquesta aplicació ens permet crear un "ordinador" dins d'un ordinador. Anem a seguir els passos d'instal·lació fins que tinga'm un escriptori amb el Sistema Operatiu que hajam instal·lat. Alguns dels SO per a descarregar:
  • [Elementary OS](https://elementary.io/es/)
  • [Ubuntu](https://ubuntu.com/download/desktop)
  • [Zorin OS](https://zorin.com/os/download/)
    •      Assignar el nom, el tipus que serà Linux i la versió que serà 64 bit perquè els ordinadors actuals tenen eixa arquitectura. Seleccionarem la memòria RAM, és a dir, la quantitat de potència que li donarem a la màquina Per últim, crearem un disc dur d'uns 10GB per la màquina que contindrà els fitxers, programes i SO. Una vegada creada la màquina, anirem a l'emmagatzematge i li carregarem el programa per instal·lar el Sistema Operatiu. Des del botó d'afegir Disc, seleccionarem el fitxer .iso que ens hem descarregat. Quan estiga carregat arrancarem la màquina que ens donarà l'opció d'instal·lar el sistema operatiu ja que detectarà que l'ordinador que hem creat està buit. Després segueix els passos d'instal·lació segons t'indique i recorda anotar-te l'usuari i contrasenya que afegisques a la màquina
