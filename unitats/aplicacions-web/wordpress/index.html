Wordpress Installation Follow this steps to install Wordpress:
  • Install virtual machine with Ubuntu 20.04 LTS
  • Install LAMP: Linux, Apache2, MySQL (server and client) and PHP
  • Set up database: sudo mysql --> create database, user, grant permission and flush
  • Download wordpress from [this link](https://wordpress.org/download/)
  • Unzip file
  • Look for config file (wp-config-sample.php) and copy to wp-config.php
  • Change the following content: DB_NAME, DB_USER, DB_PASSWORD
  • Move wordpress folder to /opt/
  • Create a symbolic link in default Apache2 path with the name wordpress and capital letters of your name (mine will be wordpress_vf). To create a symbolik link use this instruction: ln -s target link_name
  • Access to wordpress through web browser and review if you must install something else
  • When you're going to install type your full name as Site Title and remember user and password you're setting
