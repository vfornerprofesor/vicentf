Nextcloud Activity and documentation In this activity we are going to present a practical case in which your company manager asks you to investigate Nextcloud. You have to present a document explaining the following:
  • What is Nextcloud and what functions does it have (you can indicate that it is a web application).
  • How do you install it?
  • How to start using it and configure it for your company
    • To do this you have to invent a company name and what the company does in order to explain how you could use Nextcloud.
  • [System requirements](https://docs.nextcloud.com/server/latest/admin_manual/installation/system_requirements.html)
  • [Example Ubuntu installation](https://docs.nextcloud.com/server/latest/admin_manual/installation/example_ubuntu.html)
  • [Linux installation](https://docs.nextcloud.com/server/latest/admin_manual/installation/source_installation.html)

    •      Installation steps - help To explain what Nextcloud is and what it does, you can look at the documentation or other pages you can find. To do the installation you can follow the following steps (looking at the Ubuntu installation example):
  • Install Ubuntu server 22.04 LTS
  • Preconfigure: install the programs that are needed for Nextcloud to work.
  • Configure database: create database, user and give permissions
  • [Download nextcloud](https://download.nextcloud.com/server/releases/latest.zip) and NOT VERIFY
  • Put nextcloud folder in /var/www. Give permissions to www-data user
  • Configure Apache server
  • Create the nextcloud.conf file on your computer and pass it with scp on your computer (remember in alias use nextcloud_vf, where vf is your initials)
  • Enable the site with a2ensite
  • Reload apache (reload)
  • Enable mods: a2enmod rewrite, headers, env, dir, mime, ssl
  • Enable site: default-ssl
  • Reload apache (reload)
    •
  • Access the Nextcloud web application
