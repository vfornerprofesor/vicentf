Web Applications Project: develop a Web Application In this course we're going to develop a *web application* through all course. Web application will be about something you like to build, for example:
  • Task Manager
  • Expenses and income
  • Shop
  • Repairs control
    •  Web Application must includes code written in *HTML, CSS, PHP and opcionally Javascript*. It will has a *database* also to store data from users and all information about your main purpose. This work is going to be carried out in groups of 2 or 3 people. Group must write a weekly report about work and troubles you did and it will be written each week by a different member of the group. Apart of that, you are going to develop different exercises and exams *individually* to ensure your properly learning.     Blocks Web application HTML CSS PHP MySQL     Installations Apart from web application your are going to develop, you must install two web applications that are already developed and used by thousand of people. You can choose two between Mediawiki, Moodle, Wordpress and Nextcloud. You will install this web applications in a virtual machine and you must work at home. If you have troubles or doubts you can feel free to ask in class or through emails. Mediawiki Moodle Nextcloud Wordpress
